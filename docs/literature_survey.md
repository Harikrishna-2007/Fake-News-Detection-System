# Literature Survey

## Fake News Detection

Fake news detection uses Machine Learning and Natural Language Processing techniques to classify news articles as genuine or fake.

### Existing Methods

1. Logistic Regression
2. Naive Bayes
3. Random Forest
4. Support Vector Machine (SVM)

### Challenges

- Misleading headlines
- Biased datasets
- Rapid spread of misinformation

### Proposed Solution

Develop a machine learning model that predicts whether a news article is fake or real using TF-IDF and classification algorithms.
